# 🚀 AI Career Success Advisor

An AI-powered web application that helps students evaluate their academic performance, employability, and placement readiness using Machine Learning and Generative AI.

The system predicts a student's academic category, employability score, placement probability, identifies strengths and weaknesses, provides personalized recommendations, and generates an AI-powered 30-day career roadmap using Google Gemini.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Career-Success-Advisor.git
```

Move into the project

```bash
cd AI-Career-Success-Advisor
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
